# UnrealEngine-TheColiseum-ConquerorsOfRome
8K PC Game for Unreal Engine

Platform: Windows 11 x86 i64

Recommended Architecture: 
2 TB SSD; 256 GB RAM; NVIDIA RTX 5090 24 GDDR6

Charged with the murder of his family, a Roman farmer, Hebron, is charged into violent combat in the coliseum. He must become a gladiator and defeat the coliseum, to avenge their deaths.
